# RentEase - Система аренды недвижимости

## 📋 О проекте

**RentEase** - это современная веб-платформа для аренды недвижимости, автоматизирующая полный цикл сделки: от поиска объекта до подписания договора. Система обеспечивает удобное взаимодействие между арендаторами, собственниками и агентами, предоставляя интуитивно понятный интерфейс и мощный функционал.

### 🎯 Ключевые возможности

- **🔍 Поиск и фильтрация** - многофакторный поиск по цене, площади, типу недвижимости, количеству комнат; полнотекстовый поиск по адресу и названию ЖК
- **📝 Управление заявками** - полный жизненный цикл заявки: создание, обработка, одобрение/отклонение, автоматическое создание договора
- **📄 Документооборот** - автоматическая генерация договоров при одобрении заявки через триггеры PostgreSQL
- **💬 Коммуникации** - система сообщений в реальном времени с WebSocket и индикатором онлайн-статуса пользователей
- **👤 Личный кабинет** - управление профилем, аватаром, контактными данными и документами
- **📊 Аналитика** - статистика для агентов с визуализацией данных (Chart.js)
- **🏠 Управление объектами** - CRUD операции с недвижимостью, загрузка фотографий
- **🔐 Безопасность** - JWT-аутентификация, хеширование паролей, защита от SQL-инъекций

## 🏗 Архитектура

### Технологический стек

#### Backend
- **FastAPI** - высокопроизводительный асинхронный фреймворк
- **PostgreSQL 13+** - надежная реляционная СУБД
- **SQLAlchemy** - ORM для работы с базой данных
- **Pydantic** - валидация данных и управление схемами
- **JWT** - безопасная аутентификация через HttpOnly cookies
- **WebSockets** - real-time коммуникация и онлайн-статусы

#### Frontend
- **HTML5/CSS3** - адаптивная верстка с поддержкой мобильных устройств
- **JavaScript (ES6)** - интерактивность и динамическое обновление
- **Chart.js** - визуализация статистических данных
- **HTMX** - асинхронные запросы без перезагрузки страницы
